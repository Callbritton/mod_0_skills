### Class: Blender

**Attributes:**

- color (string)
- speed (integer)
- blender_on (boolean)
- lid_on (boolean)
- brand_name (string)

**Methods:**

- choose_color (new color value)
- change_speed (new speed value)
- turn_on (changes blender_on value to true)
- remove_lid (changes lid_on value to false)
